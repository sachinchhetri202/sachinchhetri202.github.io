Tic-Tac-Toe

A classic strategy game where I developed the game logic and responsive interface.

Tic-Tac-Toe is a simple yet engaging two-player game where the objective is to be the first to get three marks in a row on a 3x3 grid, either horizontally, vertically, or diagonally.

For this project, I implemented the game logic, including handling player moves, checking for winning conditions, and displaying the game outcome. The game interface is designed to be responsive and intuitive, ensuring seamless gameplay across devices.
